# Search Engine Tool

A simple search engine that crawls quotes.toscrape.com, builds an inverted index, and
lets you search for words.

## Setup

```bash
pip install -r requirements.txt
```

## Project Structure

```
src/
  crawler.py    - web crawler with 6s politeness window
  indexer.py    - inverted index builder (Stage 2)
  search.py     - search logic (Stage 3)
  main.py       - entry point
tests/
  test_crawler.py
  test_indexer.py   (Stage 2)
  test_search.py    (Stage 3)
data/
  index.json    - saved index (generated by build command)
```
